# Summary

This is a sensor node enclosure for nodeMCU ESP8266 and supporting the sensors: DHT11, TSL2561 and PIR.

The design is simple, the PIR and TSL2561 has to be glued/taped to the enclosure.  The DHT11 and ESP8266 however sits in nicely. 

Setup guide: https://techblog.calvinboey.com/esp8266_sensor_node/
